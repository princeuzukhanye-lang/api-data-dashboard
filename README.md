# API Data Dashboard

## 🚀 What it does
This project fetches real-time Bitcoin price data from an external API and displays it in a simple dashboard format.

## 🧠 Features
- Fetches live data from an API
- Displays formatted output
- Handles API errors gracefully
- Runs continuously like a real monitoring system

## 🛠 Tech Used
- Python
- REST API (Coindesk)

## 💡 Why I built this
To demonstrate how real-world applications interact with APIs and display dynamic data.

## ▶️ How to run

1. Install Python
2. Install requests:
   pip install requests

3. Run:
   python app.py
